# animation メモ

このプロジェクトでは、CSS や JS だけでは表現力に限界があったため、見出しや演出の一部を「手作業で作成したアニメーション（主に GIF）」で実装しています。

## 制作背景
- CSS/JS のみだと、繊細な手書き風・有機的な動きの再現が難しいため
- パフォーマンスと実装コストのバランスを取り、静的アニメーション（GIF/動画）を採用

## 制作と使用ツール
- 画像素素材の作成: Nano Banana / Canva（基本は無料範囲）
- 動き付け・書き出し: FlipaClip 有料版（レイヤー/書き出し機能のため）
- 形式: GIF（必要に応じて MP4 も検討）

現時点の総コスト: 約 500 円（可能な範囲は無料版で対応）

## ファイル例
- `animation/Message.gif` — メッセージ見出し用の手書き風アニメーション

## 実装メモ
- スクロールインでの再生制御: IntersectionObserver + 事前プリロードで白フラッシュ回避
- 透過背景の利用: 背景画像や下地の余白と自然に馴染ませるため
- 画質と容量のトレードオフ: ループの有無やフレーム数を調整し 1〜3MB 程度を目安に
- 配置: `index.html` 内の `.message-gif-wrapper` にて表示

## 更新手順のヒント
- 差し替えは同名ファイルで OK（キャッシュ更新が必要ならクエリ文字列を付与）
- 書き出し時はキャンバスサイズを実寸に合わせ、過剰な解像度を避ける
- 可能ならパレット最適化や色数削減でサイズを抑える

## 権利・注意
- すべて自作または AI 生成素材を組み合わせて作成
- 公開範囲や二次利用可否に注意（外部アセットを追加する場合はライセンスの確認が必要）
