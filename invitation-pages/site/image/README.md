# image メモ

このディレクトリにはヒーロー画像・背景装飾・プロフィール写真などの静的画像素材を配置します。CSS/JS だけでは質感・雰囲気（手描き感、自然な陰影等）の再現に限界があったため、AI 補助 + 手作業で素材を制作しました。

## 制作背景
- コードのみでは抽象的・記号的表現に寄りがち
- ビジュアルの没入感を高めるため、画像を主体に

## ツール / 制作フロー
- AIラフ生成: Nano Banana（構図・色味のたたき台）
- 加工・レイアウト: Canva（無料枠中心 / 必要最低限の編集）
- 微調整: 必要箇所を手動でトリミング・余白調整
- 書き出し形式: 基本は WebP 推奨 / 透過が必要な場合は PNG

現時点の総コスト: 約 500 円（可能な範囲は無料プラン利用）

## 推奨ファイルネーミング
- `hero1.png`, `hero2.png` ... スライド用
- `message_back.png` メッセージ背景
- `profile_groom.jpg`, `profile_bride.jpg` プロフィール
- 代替テキストを alt に必ず記述（アクセシビリティ配慮）

## 最適化ポリシー
- サイズは横 1600px 以内（ヒーロー用）/ プロフィールは 600px 前後
- 画像圧縮: WebP 品質 70–80 目安
- 不要な EXIF 削除（プライバシーと容量削減）
- Lazy Load（`loading="lazy"`）可能な箇所は付与

## 差し替えヒント
1. 新しい画像を同名で差し替え → ブラウザキャッシュが残る場合は `?v=2` などクエリ追加
2. 透過が不要な PNG は極力 WebP へ
3. 画質劣化が気になる場合は一段階品質を上げて再書き出し

## 注意事項 / 権利
- すべて自作または AI ツール生成素材の編集物
- 外部素材追加時はライセンス確認（商用可 / 再配布条件）
- 人物写真など個人情報性の高い画像の扱いは慎重に
